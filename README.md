dataPython ReadMe

01. python 에서 정규표현식(Regular Expression) 기능을 사용하기 위해 import re 모듈을 불러와서, 문자열 검색/패턴 검사를 실행함.

02. python 에서 collections 모듈 속 Counter 클래스를 불러와서, 데이터 개수를 자동으로 세어주는 기능을 실행함.

03. python 의 pandas 라이브러리를 활용하여 데이터프레임을 생성하고 데이터를 전처리함.
    sklearn 머신러닝 라이브러리를 활용하여 학습용(train)과 테스트용(test) 데이터셋을 분리하였으며,
    TF-IDF 기반 텍스트 벡터화를 통해 단어 중요도를 계산함. (데이터 샘플 자료 : 스팸여부)

04. TensorFlow 딥러닝 라이브러리를 활용하여 이미지 데이터를 학습 가능한 형태로 전처리한 후,
    손글씨 데이터셋(MNIST)을 CNN 모델에 적용하고, 모델 학습 및 예측을 수행함.
 
05. 감정 분석 대시보드 :
    감정에 대한 텍스트를 추출하고, 클릭 수, 시간 대를 종합적으로 분석함.
